## ðŸ†˜ Quick Help

### Can't find official source?
- Search: "site:gov.hk [topic]"
- Search: "site:pcpd.org.hk [topic]"
- Check SOURCE_CREDIBILITY_GUIDE.md for known domains

### Source behind paywall?
- Document what's visible
- Try: "[title] filetype:pdf" for free version
- Check author's institutional page
- Flag for follow-up

### Non-English content?
- Screenshot original
- Use browser translation for overview
- Note key terms in original language
- Flag for professional translation if critical

### Broken link?
- Try root domain
- Search for updated version
- Check web.archive.org
- Document as unavailable with date

### Unsure of credibility?
- Check "About Us" page
- Search organization on Wikipedia
- Cross-check with known official lists
- **When in doubt, downgrade tier**
