# Research Plan: Legal Risks in Personal Data Collection and Generative AI Deployment

## Project Overview
This research plan outlines a systematic approach to investigating legal risks associated with:
1. Collection and processing of personal data
2. Use and deployment of generative AI systems (chatbots, LLMs)
3. Related compliance requirements and regulatory frameworks

**Methodology**: Web-based research using Google Search via Playwright MCP
**Target Jurisdictions**: Hong Kong (primary), with comparative analysis of major jurisdictions
**Timeline**: Phased approach with iterative deepening

```
Tool: mcp_playwright_browser_navigate

Documentation: Record session start time in search log
```

#### Step 2: Execute Search Query
```
Tool: mcp_playwright_browser_type (search box)
Tool: mcp_playwright_browser_click (search button) OR mcp_playwright_browser_press_key (Enter)
Documentation: Log exact query string used
```

#### Step 3: Capture Search Results Page
```
Tool: mcp_playwright_browser_snapshot (accessibility snapshot)
Tool: mcp_playwright_browser_take_screenshot
Action: Save screenshot to /screenshots/YYYY-MM-DD_[topic]_google-results.png
Documentation: Record total number of results shown
```

#### Step 4: For Each Relevant Result (Top 3-5 minimum)
```
a) DOCUMENT SOURCE METADATA:
   - Complete URL (copy exact address)
   - Domain name (e.g., pcpd.org.hk, gov.hk, europa.eu)
   - Page title (as shown in search results)
   - Snippet text from Google results
   - Position in search results (#1, #2, etc.)

b) ASSESS CREDIBILITY IMMEDIATELY:
   Source Credibility Rating System:
   
   ⭐⭐⭐⭐⭐ TIER 1 - OFFICIAL/PRIMARY SOURCES (Highest Credibility)
   - .gov.hk domains (Hong Kong Government)
   - pcpd.org.hk (Privacy Commissioner for Personal Data, Hong Kong)
   - elegislation.gov.hk (Hong Kong e-Legislation)
   - europa.eu (European Union official)
   - Official regulatory body websites (.gov domains)
   - Primary legislation sources
   Example: https://www.pcpd.org.hk, https://www.elegislation.gov.hk
   
   ⭐⭐⭐⭐ TIER 2 - SEMI-OFFICIAL SOURCES (High Credibility)
   - .edu.hk / .edu domains (Universities)
   - .org domains of established regulatory bodies
   - Official guidance documents from Tier 1 sources
   - Court websites and case law databases
   - International organization official sites (OECD, UN, etc.)
   Example: https://www.hku.hk, https://caselaw.gov.hk
   
   ⭐⭐⭐ TIER 3 - PROFESSIONAL SOURCES (Moderate Credibility)
   - Major international law firm publications
   - Professional legal publishers (LexisNexis, Westlaw)
   - Industry association guidance
   - Academic journals and peer-reviewed papers
   - Reputable think tanks and research institutions
   Example: DLA Piper, Baker McKenzie, Latham & Watkins
   
   ⭐⭐ TIER 4 - NEWS/COMMENTARY (Lower Credibility - Verify)
   - Major news outlets (Bloomberg, Reuters, SCMP)
   - Legal news services (Law360, JD Supra)
   - Professional blogs by verified experts
   - Industry publications
   Example: South China Morning Post, Bloomberg Law
   
   ⭐ TIER 5 - UNVERIFIED SOURCES (Lowest - Use with Caution)
   - Unknown websites
   - Individual blogs without credentials
   - Forum posts, social media
   - Marketing content disguised as analysis
   
c) NAVIGATE TO SOURCE:
   Tool: mcp_playwright_browser_click (on search result link)
   Action: Wait for page to fully load
   Tool: mcp_playwright_browser_wait_for (if needed)

d) VERIFY PAGE LOADED CORRECTLY:
   Tool: mcp_playwright_browser_snapshot
   Check: Page title matches expectation
   Check: Content is accessible (not paywalled/blocked)

e) CAPTURE FULL SOURCE DETAILS:
   - Complete URL from address bar (may differ from search result)
   - Page title (from browser tab/page header)
   - Author/Organization name
   - Publication date (if visible)
   - Last updated date (if visible)
   - Document type (article, guidance, legislation, case law, etc.)
   
f) TAKE COMPREHENSIVE SCREENSHOTS:
   Tool: mcp_playwright_browser_take_screenshot (full page)
   Action: Save to /screenshots/YYYY-MM-DD_[topic]_[domain]_page1.png
   If multi-page: Scroll and capture additional sections
   
g) EXTRACT KEY CONTENT:
   Tool: mcp_playwright_browser_snapshot (for text extraction)
   Focus on:
   - Legal definitions
   - Regulatory requirements
   - Penalty/enforcement provisions
   - Case citations
   - Practical guidance
   - Contact information for regulatory body

h) DOCUMENT CROSS-REFERENCES:
   Note any links to:
   - Related legislation
   - Other official guidance
   - Case law citations
   - International standards referenced

i) RETURN TO SEARCH RESULTS:
   Tool: mcp_playwright_browser_navigate_back
   OR Tool: mcp_playwright_browser_tabs (if using multiple tabs)
```

### Traceability Requirements

**MANDATORY for Every Source Documented**:

1. **URL Verification**: 
   - Copy complete URL including https://
   - Test link accessibility
   - Note if link requires login/payment
   - Archive URL if possible

2. **Domain Analysis**:
   - Identify top-level domain (.gov.hk, .com, .org, etc.)
   - Determine organization ownership
   - Check "About Us" or organization info
   - Verify official status (cross-check with known official sites)

3. **Date Verification**:
   - Publication date (when content was created)
   - Last updated date (when content was modified)
   - Date accessed (when you viewed it)
   - Note if no date is visible

4. **Content Authentication**:
   - Author name/credentials (if individual)
   - Organization name (if institutional)
   - Contact information presence
   - Legal disclaimers or official seals

5. **Corroboration**:
   - Minimum 2 sources for each key finding
   - Flag single-source claims
   - Note conflicts between sources
   - Prioritize official sources for conflicts

## Quality Control Measures

### Pre-Search Quality Controls
1. **Search Query Verification**: 
   - Log exact query before executing
   - Document any query modifications
   - Note expected source types

### During-Search Quality Controls
2. **Source Verification Protocol**:
   - Cross-reference findings across minimum 2 authoritative sources
   - Verify .gov.hk sources are authentic government sites
   - Check domain registration for unfamiliar official-looking sites
   - Screenshot verification badges, official seals, contact info

3. **URL Accuracy**:
   - Copy-paste URLs, never retype manually
   - Test each URL before saving to notes
   - Note redirects or URL changes
   - Archive important pages (screenshot + text)

4. **Credibility Assessment**:
   - Assign credibility tier (1-5 stars) with written justification
   - Prioritize Tier 1-2 sources for legal requirements
   - Use Tier 3-4 for context and practical guidance
   - Flag Tier 5 sources as "unverified - corroboration required"

### Post-Search Quality Controls
5. **Currency Check**: 
   - Verify information is current (prioritize 2023-2025)
   - Flag outdated sources (pre-2020) for potential re-search
   - Note if Hong Kong PDPO amendments affect older sources
   - Track regulatory updates

6. **Jurisdiction Accuracy**: 
   - Ensure Hong Kong-specific requirements clearly separated
   - Tag all sources by jurisdiction
   - Note extraterritorial application (GDPR, CCPA on HK entities)
   - Identify conflicts between jurisdictions

7. **Completeness Review**: 
   - Check all planned search queries executed (use tracker checkboxes)
   - Verify minimum source requirements met per query
   - Confirm Tier 1 source captured for each major topic
   - Document searches that yielded no useful results

8. **Traceability Audit**:
   - Every source has complete URL
   - Every source has credibility rating
   - Every source has date information
   - Every source has screenshot evidence
   - Every finding traceable to specific source

9. **Gap Analysis**: 
   - Identify areas needing additional research
   - Note topics with insufficient official guidance
   - Flag areas where only Tier 3-5 sources available
   - Plan follow-up searches for gaps
